🏎️ Assetto Corsa Linux Setup (NixOS Fork)

这是我基于原仓库 Fork 修改的版本。增加了 nix-shell shebang 支持，去除了写死的 apt/pacman 依赖检查，能够自动在隔离环境中拉取运行所需依赖（wget, unzip, protontricks 等），完美适配 NixOS 系统。

Fork updated by Jaanai-Liu: Adds support for NixOS using the Nix package manager via nix-shell shebang to automatically pull dependencies in an isolated environment.

功能特性 (What it does)

Features

  • 自动安装 Proton-GE
  • 安装 Content Manager 并配置环境
  • 安装 Custom Shaders Patch (CSP)
  • 安装 DXVK 以提升图形性能
  • 将 Content Manager 添加至 mimeapps (支持网页直接拉起服务器连接)
⚠️ 前置要求 (Crucial First Step)

在运行脚本之前，你必须通过 Steam 至少运行一次原版游戏（使用内置的任意 Proton 版本），以生成 compatdata 前缀文件夹。 You must launch the vanilla game at least once via Steam (using any built-in Proton) to generate the compatdata prefix before running this script.

一键安装指令

Installation Command

curl -Os https://raw.githubusercontent.com/Jaanai-Liu/assettocorsa-linux-setup/main/assettocorsa-linux-setup.sh && chmod +x assettocorsa-linux-setup.sh && ./assettocorsa-linux-setup.sh
