低内存及受限网络 VPS 上 NixOS Anywhere 安装指南

NixOS-Anywhere Installation Guide for Low-RAM & Network-Restricted VPS

本指南提供了一套针对低内存 VPS（如 1GB 内存）及连接 GitHub 网络较差的机器（如国内阿里云、腾讯云）安装 NixOS 的完美工作流。本方案通过分步执行、利用本地算力代工，以及在内存系统中开启 ZRAM 虚拟内存或手动推送镜像，完美解决了卡死问题。

This guide provides a bulletproof workflow for installing NixOS on low-memory VPS (e.g., 1GB RAM) and servers with restricted network access to GitHub. It solves OOM errors and network timeouts by splitting the installation into phases, utilizing local computation, and creating a temporary ZRAM swap space.

方法一：标准 kexec 流程

Method 1: Standard kexec

适用于连接 GitHub 网络良好的海外 VPS。

For overseas servers with good GitHub connection.

第一步：准备工作与 SSH 密钥管理

Step 1: Preparation & SSH Key Management 

ssh-keygen -R <YOUR_VPS_IP>

第二步：注入 NixOS 内存系统 (kexec 阶段)

Step 2: Inject NixOS Memory System

nix run github:nix-community/nixos-anywhere -- --phases kexec --flake .#<YOUR_FLAKE_HOST> root@<YOUR_VPS_IP> --no-substitute-on-destination

第三步：开启 ZRAM 极限扩容（可选但强烈建议）

Step 3: Enable ZRAM Extreme Expansion (Recommended) 

mount -o remount,size=2G /nix/.rw-store
modprobe zram
echo zstd > /sys/block/zram0/comp_algorithm
echo 1500M > /sys/block/zram0/disksize
mkswap /dev/zram0
swapon /dev/zram0
exit

第四步：执行硬盘分区与系统安装

Step 4: Execute Disk Formatting and Installation 

nix run github:nix-community/nixos-anywhere -- --phases disko,install --flake .#<YOUR_FLAKE_HOST> root@<YOUR_VPS_IP> --no-substitute-on-destination

方法二：手动 kexec 流程 (夺舍)

Method 2: Manual kexec (For Domestic Servers)

适用于国内云服务器等连接 GitHub 极慢的 VPS。避开 `Scheme missing` 报错。

For servers in China with poor GitHub connections. Avoids Scheme missing errors.

第一步：在本地下载 kexec 镜像文件

Step 1: Download the kexec image locally 

curl -L https://mirror.ghproxy.com/https://github.com/nix-community/nixos-images/releases/download/nixos-25.05/nixos-kexec-installer-noninteractive-x86_64-linux.tar.gz -o nixos-kexec.tar.gz

第二步：通过 SCP 将镜像手动推送到服务器

Step 2: SCP the image to the VPS

scp ./nixos-kexec.tar.gz root@<YOUR_VPS_IP>:/root/

第三步：SSH 登录并手动触发 kexec

Step 3: Trigger kexec manually via SSH 

ssh root@<YOUR_VPS_IP> "mkdir -p /root/kexec && tar -zxf /root/nixos-kexec.tar.gz -C /root/kexec && /root/kexec/kexec/run"

第四步：执行硬盘分区与系统安装

Step 4: Execute Disk Formatting and Installation 

nix run github:nix-community/nixos-anywhere -- \
  --flake .#<YOUR_FLAKE_HOST> \
  --phases disko,install \
  --no-substitute-on-destination \
  root@<YOUR_VPS_IP>
