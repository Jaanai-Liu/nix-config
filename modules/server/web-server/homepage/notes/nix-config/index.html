❄️ My NixOS Configuration

包含了我的个人模块化 NixOS 配置文件。基于 Niri 窗口管理器、Home-manager 以及 Agenix 进行构建。

Contains my personal, modularized NixOS configurations built with Niri, Home-manager, and Agenix.

🧩 核心软件栈

🧩 Software Stack

  • 窗口管理器 (WM): Niri (滚动平铺 Wayland 合成器)
  • 密钥管理: Agenix (保护 Wi-Fi 密码与 API keys)
⚠️ Critical Warnings

请勿直接部署此 Flake。它包含需要我的私钥才能解密的 Agenix 密钥。 Do not deploy this flake directly. It contains encrypted secrets managed by Agenix.
