## 行程概览

逆时针环伊犁，跳过那拉提，从特克斯经尼勒克直达唐布拉，省 1.5 天车程。6 月草原野花与薰衣草双花期，节奏宽松不赶路，双司机轮换。全程约 1700km（含返程）。

| 天数  | 路线                               | 车程          | 亮点                     |
| ----- | ---------------------------------- | ------------- | ------------------------ |
| Day 1 | 抵达乌鲁木齐                       | —             | 休整、逛吃市区           |
| Day 2 | 乌鲁木齐 → 赛里木湖                | 550km / 6.5h  | 赛湖环湖、果子沟大桥日落 |
| Day 3 | 赛里木湖 → 霍城 → 伊宁             | 190km / 3.5h  | 薰衣草花海、喀赞其民俗村 |
| Day 4 | 伊宁 → 特克斯 → 喀拉峻草原         | 170km / 3h    | 立体草原、阔克苏大峡谷   |
| Day 5 | 特克斯 → 尼勒克 → 唐布拉百里画廊   | 270km / 5.5h  | 恰甫其海水库、百里画廊   |
| Day 6 | 唐布拉深度游                       | 0             | 仙女湖、喀什河河谷       |
| Day 7 | 唐布拉 → 独库北段 → 乌鲁木齐       | 440km / 9-11h | 独库公路全地貌穿越       |
| Day 8 | 乌鲁木齐 → 返程                    | —             | 博物馆、大巴扎           |

## 出行提示

**天气** — 6 月气温 12~28℃，早晚温差大。必备：防风外套、长裤、防晒霜、墨镜、帽子。

**独库公路** — 6 月中下旬开通，出发前一周关注 **"平安独库"公众号** 确认实时路况。旺季（尤其乔尔玛至那拉提段）实行分时段分路段预约通行，需在 **"新疆交通公众出行"小程序** 提前预约。

**限速** — 新疆全程区间测速，严格按限速行驶，勿疲劳驾驶。

**住宿** — 6 月旺季，热门地点（赛里木湖、唐布拉）提前 1 个月预订。

**时差** — 新疆使用北京时间，但日出约 6:30、日落约 22:00，游玩时间充裕。

## Day 2 — 乌鲁木齐 → 赛里木湖

全天以赶路为主，下午抵达赛湖环湖，傍晚拍果子沟大桥日落。双司机轮换无压力。

| 时间        | 行程               | 备注                                   |
| ----------- | ------------------ | -------------------------------------- |
| 07:30       | 乌鲁木齐出发       | G30 连霍高速一路向西                   |
| 12:00       | 精河服务区午餐休整 | 停留 1 小时                            |
| 14:30       | 抵达赛里木湖景区   | 购票自驾环湖（门票约 145/人）          |
| 15:00-17:30 | 顺时针环湖 90km    | 6 月湖畔野花盛开，重点拍草原、雪山倒影 |
| 18:00       | 果子沟观景台       | 拍摄果子沟大桥日落                     |

**车程** — 550km，全程高速，约 6.5 小时

**路线** — 乌鲁木齐 → G30 连霍高速 → 赛里木湖

### 住宿方案二选一（后续讨论决定）

> **原方案问题**：原计划 Day 2 住清水河镇，Day 3 又安排回赛湖看清晨——清水河距赛湖约 60km，来回多跑 120km 冤枉路。

**方案 A / 住赛湖附近（推荐）**

住赛里木湖南门或东门附近毡房 / 简易酒店。Day 3 清晨自然出发，零绕路。缺点是住宿条件较简陋、旺季价格高（约 400-800/晚）。

**方案 B / 住清水河镇**

住清水河镇酒店，条件好、性价比高（约 200-400/晚）。但 Day 3 需放弃赛湖清晨环节，直接从清水河出发经果子沟大桥去霍城。

**美食** — 新疆拌面、大盘鸡

## Day 4 — 伊宁 → 特克斯 → 喀拉峻草原

| ----------- | ------------------------------ | --------------------------------------------------------- |
| 08:00       | 伊宁出发                       | —                                                         |

| 16:00-18:30 | 阔克苏大峡谷                   | 人体草原、鳄鱼湾、九曲十八弯                              |

## Day 7+8 — 唐布拉 → 乌鲁木齐 → 返程

能否走独库北段需要出发前确认。以下三个方案，对应三种不同路线。

| 方案 | 条件 | 路线 | 路程 | Day 7 终点 | Day 8 |
|------|------|------|------|-----------|-------|
| A | 独库通车 | 唐布拉 → 独库北段 → 独山子 → 乌市 | 440km | 乌市（下午到） | 乌市休闲 + 返程 |
| B | 独库不通 | 唐布拉 → 乔尔玛 → G218 → 巴仑台 → 乌市 | ~700km | 乌市（晚上到） | 乌市休闲 + 返程 |
| C | 独库不通 | 唐布拉 → 伊宁 → G30 高速 → 奎屯 | ~950km | 奎屯（住一晚） | 上午到乌市 + 返程 |

---

### 方案 A（首选）：独库通车，穿越北段

一天从唐布拉玩回乌市，独库北段本身就是风景。

| 时间        | 行程                           | 备注                                                      |
| ----------- | ------------------------------ | --------------------------------------------------------- |
| 06:30-08:00 | 唐布拉 → 乔尔玛 → 进入独库北段 | S315 + G217                                               |
| 08:00-12:00 | 独库北段穿越                   | 哈希勒根达坂、天瀑、守望天山碑，2-3 个观景台各停 10-15min |
| 12:00-13:00 | 独山子午餐                     | 休整 1 小时                                               |
| 13:00-16:00 | 独山子 → 乌鲁木齐              | G30 连霍高速                                              |
| 16:30       | 抵达乌鲁木齐市区               | 自由活动                                                  |

| 路段              | 距离   | 时间  | 路况                               |
| ----------------- | ------ | ----- | ---------------------------------- |
| 唐布拉 → 乔尔玛   | ~40km  | 45min | S315 河谷路段                      |
| 乔尔玛 → 独山子   | ~150km | 4-5h  | 独库北段核心，山路盘旋，限速 40-60 |
| 独山子 → 乌鲁木齐 | ~250km | 3h    | 全程高速                           |

**Day 8**：乌市全天自由，博物馆、大巴扎、美食，下午或晚上航班从容返程。

---

### 方案 B：独库不通，走 G218 国道经巴仑台

> 路线：唐布拉 → 乔尔玛 → 那拉提 → G218 国道 → 巴仑台 → 和静 → 托克逊 → G30 高速 → 乌市

独库北段封了不代表中段也封。只要乔尔玛↔那拉提一段还能走，就可以从那拉提转 G218 一路向东，翻过天山南麓经巴仑台回乌市。**比原路返回少 250km，Day 7 当晚就能到乌市**，缺点是 G218 山区国道弯多路窄。

| 时间        | 行程                  | 备注                        |
| ----------- | --------------------- | --------------------------- |
| 06:30-08:00 | 唐布拉 → 乔尔玛       | S315，45min                  |
| 08:00-09:30 | 乔尔玛 → 那拉提       | 独库中段，需确认是否通车    |
| 09:30-15:30 | 那拉提 → G218 → 巴仑台 | 巩乃斯河谷，午饭路边解决    |
| 15:30-18:30 | 巴仑台 → 和静 → 托克逊 | 峡谷 + 干沟                 |
| 18:30-20:30 | 托克逊 → 乌鲁木齐     | G30 高速                    |
| 21:00       | 抵达乌市，入住        | —                           |

| 路段                   | 距离   | 时间  | 路况                        |
| ---------------------- | ------ | ----- | --------------------------- |
| 唐布拉 → 乔尔玛        | ~40km  | 45min | S315 河谷                   |
| 乔尔玛 → 那拉提        | ~60km  | 1.5h  | 独库中段，需确认通车         |
| 那拉提 → 巴仑台        | ~280km | 5-6h  | G218 国道，巩乃斯河谷，弯多 |
| 巴仑台 → 和静 → 托克逊 | ~180km | 3h    | 峡谷路段 + 干沟             |
| 托克逊 → 乌鲁木齐      | ~160km | 2h    | G30 高速                    |

**合计** — 约 700km，含休息约 14-15h。6:30 出发，21:00 左右到乌市。虽然累，但 Day 8 不用赶路，睡到自然醒再返程。

**Day 8**：乌市全天自由——晚起、吃顿好的、大巴扎扫货，下午或晚上航班从容返程。

---

### 方案 C：独库不通，原路返回走 G30 高速

> 路线：唐布拉 → 尼勒克 → 伊宁 → G30 连霍高速 → 奎屯

独库北段和中段都不通（或不想走 G218 烂国道）时，沿原路经伊宁折返。高速占比高，驾驶比 G218 轻松，但 950km 太远，**必须在奎屯住一晚**，是三个方案中唯一 Day 7 到不了乌市的。

| 路段                   | 距离   | 时间   | 路况             |
| ---------------------- | ------ | ------ | ---------------- |
| 唐布拉 → 尼勒克 → 伊宁 | ~280km | 4.5h   | 国道山路         |
| 伊宁 → 奎屯            | ~450km | 5h     | G30 高速，路况好 |

Day 7 约 730km，含休息约 10h，住奎屯。

**Day 8：奎屯 → 乌鲁木齐 → 返程**

| 时间        | 行程        | 备注         |
| ----------- | ----------- | ------------ |
| 09:00-12:00 | 奎屯 → 乌市 | 250km / 2.5h |
| 12:30-14:00 | 午饭        | —            |
| 14:30       | 前往机场    | —            |

航班选 Day 8 16:00 以后。
